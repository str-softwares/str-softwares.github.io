What is my task pool?

My Task Pool is a service that records your tasks! After signing up for an account, users can log their tasks and assign them scores. Every task contains 3 scores: Impact, Ease and Confidence. Each task can be added, edited and deleted. All the tasks will be sorted by the average of all three scores in descending order (i.e., 8, 5, 4). Every API call will return 10 tasks. When not using the service, users can log out of the system.

Important:
Allowed actions to user via REST API (No front-end needed):

  • User should be able to register and login to the system.
  • User should be able to add, edit and delete tasks.
  • User should be able to view all tasks sorted by average score.
  • User should be able to logout from the system.
API Response Format:
The API is organized around REST. Our API has predictable, resource-oriented URLs, and uses HTTP response codes to indicate API errors. JSON is returned by all API responses, including errors.

Response Codes:
HTTP response codes to indicate the success or failure of an API request. In general, codes in the 2xx range indicate success, codes in the 4xx range indicate an error that failed given the information provided (e.g., a required parameter was omitted, a charge failed, etc.), and codes in the 5xx range indicate an error with servers (these are rare).
